# Проект: Линейная и логистическая регрессия с нуля

**Автор:** Гаськов Валерий

---

## Описание проекта
В проекте реализованы:

- Линейная регрессия с градиентным спуском (batch и mini-batch)  
- Логистическая регрессия с градиентным спуском (binary и multiclass)  
- Классификация с дополнительной моделью Random Forest  
- Метрики: Accuracy, Precision, Recall, F1-score, Confusion Matrix, ROC AUC (для бинарной классификации)  
- Эксперименты с различными learning rate, epoch и batch size  
- Интерактивный интерфейс с виджетами для выбора модели и параметров обучения

---

## Датасет
Файл: `master.csv`  
Источник: [Suicide Rates Overview 1985 to 2016 — Kaggle](https://www.kaggle.com/datasets/russellyates88/suicide-rates-overview-1985-to-2016/discussion/570568)  

Основные используемые колонки:  
- **sex** — бинарная зависимая переменная (мужчина / женщина)  
- **age** — мультиклассовая зависимая переменная  
- Другие признаки: GDP, population и т.д.
